* [drpyS](https://github.com/hjdhnx/drpy-node)
* [zyplayer](https://github.com/Hiram-Wong/ZyPlayer)
* [海阔$工具](https://www.yuque.com/u537126/kb/oq16z4?singleDoc#《海阔视界$工具文档》 )
* [海阔视界](https://docs.189.tyrantg.com/)
